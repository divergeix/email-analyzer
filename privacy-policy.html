Privacy Policy

Divergeix Email Analyzer — Chrome Extension

Last updated: April 2026

Overview

Divergeix Email Analyzer is a browser extension that helps users analyze and summarize Gmail email threads. We are committed to protecting your privacy and being transparent about how the extension works.

Data Collection

This extension does not collect, store, or transmit any personal data to our servers. We do not have any servers. All processing happens locally in your browser or via direct API calls from your browser.

What Data the Extension Accesses

  • Email content: The extension reads email thread content (sender names, dates, and message bodies) from the currently open Gmail thread — only when you click "Run Full Analysis" or use other features. This data is never stored on any external server.
  • API Key: Your API key is stored locally in your browser's storage (chrome.storage.sync). It is never shared with anyone.

How Email Content Is Processed

  • When you analyze an email, the thread content is sent directly from your browser to the AI API for processing.
  • The API processes the content and returns the analysis results.
  • No intermediary servers are involved — the call goes directly from your browser to the API.
  • We do not log, store, or have access to any email content or API responses.

Local Storage

  • API Key: Stored in chrome.storage.sync (encrypted by Chrome).
  • Theme preference: Light/dark mode choice stored in chrome.storage.local.
  • Analysis history: Last 5 analyses stored locally in chrome.storage.local for your convenience. You can clear this at any time from the History tab.

Third-Party Services

  • The extension makes API calls to process email content. These calls are made directly from your browser using your own API key.
  • Google Calendar links are generated locally — no data is sent until you choose to click the link.

Permissions Explained

  • mail.google.com: Required to read email thread content from Gmail for analysis.
  • storage: Required to save your API key and preferences locally.
  • clipboardWrite/Read: Required for copy-to-clipboard features (Copy Reply, Copy as WhatsApp, etc.).

Data Retention

All data is stored locally in your browser. Uninstalling the extension removes all stored data. No data is retained on any external server.

Children's Privacy

This extension is not directed at children under the age of 13.

Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected on this page with an updated date.
